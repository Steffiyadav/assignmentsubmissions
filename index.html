File support

Version control

Multiple uploads per assignment — restore previous submissions when needed.

Receipt & audit

Each upload generates a timestamped submission receipt for record-keeping.

How it works

  1. Student uploads assignment file and enters course & assignment name.
  2. System stores file, creates a submission receipt and allows optional overwrite to create new version.
  3. Instructor can download submissions or leave feedback (requires server-side features).
This static site is intended as a landing/front-end. For full functionality (PHP/DB), server hosting is required.
